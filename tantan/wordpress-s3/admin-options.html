Amazon S3 Plugin for WordPress

Plugin Updates:
Check for updates to this plugin >

This plugin allows you to easily use objects stored on Amazon's S3 web service.

Installation and Usage: Just follow the onscreen prompts to link this plugin to your Amazon S3 account. Once your account is linked, you will see an "Amazon S3" tag in your edit screen.
